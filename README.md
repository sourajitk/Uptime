# Uptime

> Easily check your computer's uptime straight from your taskbar!

Uptime is a lightweight and theme-adaptive Windows utility that displays your system's exact running uptime directly in the system tray.

<p align="center">
  <img src="https://i.imgur.com/y9VLM4b.png" alt="Uptime Screenshot" />
</p>

## Features

Uptime is super easy to use! Just run the installer, and you can monitor your computer's uptime in your system tray immediately.

## Installation

Download the latest installer (`UptimeTaskbarAppSetup.exe`) from the [Releases](https://github.com/sourajitk/Uptime/releases) page.
